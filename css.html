CSS

CSS se encarga del diseño y la apariencia visual de una página web.

¿Qué es CSS?

CSS permite controlar colores, tamaños, fuentes y la distribución de los elementos en una página web.

Ventajas

Mejor organización

Permite separar contenido (HTML) del diseño (CSS).

Reutilización

Los estilos se pueden aplicar en varias páginas fácilmente.

Mejor experiencia

Ayuda a que el sitio se vea más claro y agradable.
