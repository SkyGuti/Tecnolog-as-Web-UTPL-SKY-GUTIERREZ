HTML

HTML es el lenguaje base para crear la estructura de una página web.

¿Qué es HTML?

HTML permite estructurar el contenido usando etiquetas como títulos, párrafos, imágenes y enlaces.

Ventajas

  • Fácil de aprender
  • Compatible con todos los navegadores
  • Base del desarrollo web
