¿Qué es HTML y para qué sirve?

HTML es el lenguaje estándar para crear páginas web. No programa acciones, sino que define qué es cada elemento: un título, un párrafo, una imagen, una lista, un enlace, entre otros.

Gracias a HTML el navegador entiende cómo organizar el contenido en la pantalla y puede mostrar la estructura completa de un sitio web.

Estructura básica de una página HTML

Aunque cada página puede ser diferente, casi todas comparten una estructura general. De manera sencilla, se puede resumir en las siguientes partes:

  1. Declaración DOCTYPE: indica que el documento utiliza la versión HTML5.
  2. Elemento html: agrupa todo el contenido de la página.
  3. Sección head: contiene el título de la pestaña y los metadatos.
  4. Sección body: incluye los textos, imágenes, enlaces y demás elementos visibles.

Etiquetas importantes en HTML

En una página web se usan muchas etiquetas, pero algunas aparecen con mucha frecuencia. Entre las más utilizadas se encuentran:

  • Etiquetas de título: h1, h2, h3 y otras para los encabezados del contenido.
  • Etiqueta p: se usa para escribir párrafos de texto.
  • Etiqueta a: permite crear enlaces hacia otras páginas o sitios web.
  • Etiqueta img: inserta imágenes dentro del documento.
  • Etiquetas de lista: ul, ol y li para crear listas de elementos.

¿Dónde se utiliza HTML en la vida real?

HTML se usa en casi cualquier sitio web que visitamos a diario. Algunos ejemplos de dónde aparece son:

  • Páginas informativas y blogs personales.
  • Portales de noticias y sitios educativos.
  • Tiendas en línea y catálogos de productos.
  • Interfaces básicas de aplicaciones web y móviles híbridas.

En todos estos casos HTML se combina con CSS y JavaScript para formar experiencias completas y fáciles de usar para el usuario.
