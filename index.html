Lenguajes y funciones

Para construir una página web se combinan tres partes: estructura, diseño e interactividad. HTML define la estructura y el contenido de la página; CSS se encarga del aspecto visual (colores, fuentes y organización); y JavaScript añade comportamiento e interactividad, como respuestas a clics, mensajes y validación de formularios.
