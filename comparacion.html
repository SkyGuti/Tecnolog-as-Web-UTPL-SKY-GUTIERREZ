Tabla comparativa de sus roles en el desarrollo web

La siguiente tabla resume el papel principal de cada lenguaje, el tipo de lenguaje que representa y algunos ejemplos sencillos de cómo se utiliza dentro de una página web.

Lenguaje Rol principal Tipo de lenguaje ¿Qué hace? Ejemplo de uso
HTML Define la estructura del contenido que verá el usuario en la pantalla. Lenguaje de marcado orientado a organizar información. Indica qué partes son títulos, párrafos, imágenes, listas o enlaces. Crear una página con un título, un texto de presentación y una imagen.
CSS Controla la apariencia visual de los elementos definidos por HTML. Lenguaje de hojas de estilo en cascada. Cambia colores, tipos de letra, tamaños, márgenes y posición de los bloques. Dar un color de fondo a la página y centrar los títulos en la parte superior.
JavaScript Añade interactividad y comportamiento a la página web. Lenguaje de programación ejecutado en el navegador. Responde a clics del usuario, modifica textos e imágenes y realiza cálculos. Mostrar un mensaje de bienvenida cuando el usuario presiona un botón.
