La siguiente tabla muestra el rol que cumple cada lenguaje dentro del desarrollo web y qué aporta a una página.

Lenguaje Rol en el desarrollo web ¿Qué aporta al sitio?
HTML Estructura del contenido. Define títulos, párrafos, imágenes, listas y enlaces que forman la página.
CSS Diseño y presentación. Controla colores, fuentes, tamaños y la organización visual de los elementos.
JavaScript Interactividad y comportamiento. Permite responder a acciones del usuario y actualizar el contenido en pantalla.
